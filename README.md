# next-admin
a project about admin ,use next antd redux 
### cd next-admin
    npm i
    npm run dev 开发 （配置接口代理在routers/api urllist）

    npm run build 打包
    npm run start 生产

    npm run build 打包
    npm run start-test 测试

    pm2部署
    npm run build
    pm2 start ecosystem.config.js --env production
